# Assets de Imagens

## Estrutura

- `before.PNG` / `after.PNG` - Imagens do comparador antes/depois (já existem)
- `whatsapp-background-pattern.png` - Padrão de fundo do WhatsApp (adicionar)
- `services/` - Imagens do carrossel de serviços (adicionar)

## Imagens Necessárias

### WhatsApp Background Pattern
- **Arquivo:** `whatsapp-background-pattern.png`
- **Uso:** Background da janela do chat WhatsApp
- **Tamanho sugerido:** 200x200px (pode ser repetido/tile)
- **Formato:** PNG com transparência ou SVG

### Imagens de Serviços
Adicionar na pasta `services/`:
- `service-tecnologia.jpg` - Equipamentos/tecnologia
- `service-agilidade.jpg` - Velocidade/agilidade
- `service-protecao.jpg` - Proteção/blindagem
- `service-garantia.jpg` - Garantia/confiança

**Tamanho sugerido:** 1200x800px (aspect ratio 3:2)
**Formato:** JPG ou WebP (otimizado)
